# Day 02 Data Structures

## Overview

This folder contains my practice exercises completed during **Day 02** of the internship training plan. The focus of this day was understanding and implementing Python data structures and improving problem-solving skills using lists, tuples, dictionaries, sets, and strings.

---

## Topics Covered

### Lists

* Indexing
* Slicing
* Common Methods:

  * `append()`
  * `extend()`
  * `insert()`
  * `remove()`
  * `pop()`
  * `sort()`
* List Comprehensions

### Tuples

* Immutability
* Packing and Unpacking
* When to use tuples instead of lists

### Dictionaries

* Dictionary Creation
* Methods:

  * `get()`
  * `keys()`
  * `values()`
  * `items()`
  * `update()`
* Dictionary Comprehensions

### Sets

* `union()`
* `intersection()`
* `difference()`
* `symmetric_difference()`
* Common Use Cases

### Strings

* `split()`
* `join()`
* `strip()`
* `replace()`
* `find()`
* `count()`
* String Immutability
